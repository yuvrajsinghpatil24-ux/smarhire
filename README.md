# SmartHire

A job portal web app built with React + Vite + TypeScript + Tailwind CSS + shadcn/ui.

## How to Run Locally

```bash
npm install
npm run dev
```

## How to Deploy on Vercel

1. Go to https://vercel.com
2. Click "Add New Project"
3. Upload this folder or connect your GitHub repo
4. Framework: Vite (auto-detected)
5. Click Deploy ✅

## Tech Stack
- React 18
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Router DOM
- Framer Motion
- TanStack Query
